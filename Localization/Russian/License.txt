4. Политика конфиденциальности

4.1. Собираемая и отправляемая информация
При установке приложение создает уникальный идентификатор в виде случайным образом сгенерированного числового кода (далее UID). UID генерируется таким образом, чтобы информация, на основе который он был создан, никак не могла быть восстановлена. UID отправляется на наш сервер в момент запроса наличия обновлений для данного ПО, и используется для определения количества активных пользователей (популярности приложения). Помимо этого, на сервер обновлений отправляется информация об используемой локализации, версии ПО "MouseMAC", версии операционной системы и ее разрядности. Эта информация необходима для того, чтобы определить наличие обновлений ПО MouseMAC для данного языка и версии операционной системы. Вы можете отказаться от передачи этой информации, путем отключения функции автоматической проверки обновлений в настройках программы.
